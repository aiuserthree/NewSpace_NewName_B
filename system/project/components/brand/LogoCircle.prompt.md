**LogoCircle** — a 48px round white badge for a third-party app logo in an integration row; the one place literal brand colours appear.

```jsx
<LogoCircle label="Slack"><img src="/assets/slack.svg" width="24" /></LogoCircle>
<LogoCircle>A</LogoCircle>
```

- `size` sets the diameter. Pass an `<img>`, letter, or Icon as children. Arrange several in a row/grid to show ecosystem breadth.
