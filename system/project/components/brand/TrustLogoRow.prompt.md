**TrustLogoRow** — a grayscale partner-logo strip under a small uppercase label, for social proof.

```jsx
<TrustLogoRow label="TRUSTED BY 70,000+ COMPANIES" logos={["Acme", "Globex", "Umbra", "Initech"]} />
```

- Logos render in ink grayscale at reduced opacity so the orange stays uncontested. String entries become ghosted wordmarks; pass nodes for real marks.
